# Handoff: 명운봄 사주 구매 퍼널 (밤하늘 붓글씨 무드)

## Overview
명운봄(命運春)은 사주 분석 구매 퍼널입니다. 무료 기본 분석으로 신뢰를 만들고 → ₩6,900 단일 결제 → 결제 직후 ₩19,900 업셀 → 같은 사주 데이터로 멀티 SKU 재구매(LTV 루프)로 이어지는 모바일 퍼스트 서비스입니다.

흐름: **랜딩 → 카카오 로그인 → 정보·고민 입력 → 무료 기본 분석 → 결제(₩6,900) → 업셀(₩19,900) → 완료·전달 → 추가 SKU 재유입 ↻**

## About the Design Files
이 번들의 `.dc.html` 파일들은 **HTML로 만든 디자인 레퍼런스(프로토타입)**입니다 — 의도한 외형·동작을 보여주는 시안이지, 그대로 복사해 배포할 프로덕션 코드가 아닙니다.

작업의 목표는 이 HTML 디자인을 **대상 코드베이스의 기존 환경(여기서는 PRD §11 기준 Next.js + Vercel)에서 그 환경의 패턴·라이브러리로 재구현**하는 것입니다. 아직 코드베이스가 없다면 프로젝트에 가장 적합한 프레임워크를 골라 구현하면 됩니다. HTML/인라인 스타일을 그대로 옮기지 말고, 아래 디자인 토큰과 스펙을 기준으로 컴포넌트화하세요.

> 참고: `.dc.html`은 "Design Component" 포맷이라 `support.js` 런타임이 있어야 브라우저에서 열립니다. 구현 시에는 무시하고 마크업/스타일/카피만 참고하세요.

## Fidelity
**High-fidelity (hifi)** — 최종 색상·타이포·간격·인터랙션이 들어간 목업입니다. 아래 토큰의 정확한 hex/px 값을 그대로 사용해 픽셀에 가깝게 재현하세요. (단, 배경 야경은 CSS/SVG로 그린 플레이스홀더이며, 실제 야경 사진/일러스트로 교체 가능 — Assets 참고.)

와이어프레임(`사주 퍼널 와이어프레임.dc.html`)만 **low-fidelity**로, 구조·플로우 참고용입니다.

### 0. 데스크톱 랜딩 히어로 (`명운봄 히어로.dc.html`)
- **Purpose**: 첫 진입 후킹. 풀스크린 야경 + 브랜드.
- **Layout**: 1920×900 풀블리드. 좌상단 초승달 로고(SVG), 상단 중앙 명조 메뉴(명운봄 소개 / 사주 의뢰신청 / 의뢰인 문의), 우상단 로그인·회원가입, 중앙 상단 붓글씨 금빛 워드마크 "명운봄 (命運春)" + 부제 "달빛 아래, 당신의 운을 읽는 곳", 우측 큰 보름달(300px, 따뜻한 금빛 radial + glow), 한옥 처마 실루엣(우측 SVG), 앙상한 나뭇가지(상단 SVG), 좌하단 붓글씨 헤드라인 + 명조 리드, 01/02 슬라이드 인디케이터, 우하단 카카오 노란 알약 "카톡 문의".
- **헤드라인**: 명조 리드 "마음이 어지러운 밤," (30px·700·`#D8D2C4`) → 붓글씨 86px `#F4F1E8` "**당신**의 사주가 / 달빛 아래 **펼쳐집니다**" (키워드 "당신"·"펼쳐집니다" = `#E7C27D` 금색).
- **배경**: CSS 그라데이션 야경 + SVG 달/처마/가지/산 실루엣 + 반짝이는 별(twinkle 4s). 실제 사진으로 교체 가능.

### 1. 랜딩 (모바일) ①
- **Purpose**: 3초 후킹 + 무료 분석 유도.
- **Layout**: 세로 스택 — 히어로(워드마크·달 모티프·붓글씨 헤드라인·금 CTA·평점) → 구분선 → "이런 이야기를 들려드려요"(壹貳參 + 항목) → 후기 카드 → 환불 보장 문구.
- **핵심 카피**: 헤드라인 붓글씨 "올해, 당신의 사주는 / 무엇을 말할까요". CTA(금 그라데이션) "내 사주 무료로 확인하기". 평점 "★★★★★ 4.9 · 누적 12,800명이 받았어요". 환불 "제작·발송 전 100% 환불 보장"(`#6FBE8B`).
- 달 모티프: 96px 원, `radial-gradient(circle at 38% 35%,#3A2E5E,#171430)` + 글로우.

### 2. 카카오 로그인 ②
- **Purpose**: 가입 마찰 제거 + 재방문/데이터 락인(멀티 SKU 재유입 기반).
- **Layout**: 초승달 로고 → 붓글씨 워드마크 → 붓글씨 헤드라인 "3초 만에 시작하고 / 내 사주 받기" → 카카오 노란 버튼(아이콘+텍스트) → "재방문이면 입력 정보를 자동으로 불러와요" → 구분선 → 동의 체크 2개([필수] 개인정보 수집·이용 / [필수] 제3자 제공) → 약관 고지.
- 체크박스(체크됨): 18px, `#E7C27D` 배경 + 먹색 체크 SVG.

### 3. 정보·고민 입력 ③
- **Purpose**: 사주 산출 정보 + **고민 자유텍스트(개인화 핵심)** 수집.
- **Layout**: 상단 진행바(금 그라데이션 66%) → 붓글씨 "사주 정보를 / 알려주세요" → 폼: 이름(선택, 플레이스홀더 "비우면 '회원님'으로 불러드려요") · 성별 라디오(남/여, 선택=금 테두리+틴트) · 생년월일(달력 아이콘) · 양/음력 토글(+"음력 시 윤달 체크") · 태어난 시각 select(+"모름" 칩) → **고민 textarea(보라 강조 박스 `rgba(138,107,242,.1)` + `#6541F2` 테두리, 라벨 "고민 내용 ★ 개인화 핵심", 플레이스홀더 예시)** → [필수] 동의 → 금 CTA "무료 분석 결과 보기".
- 필드 상세는 PRD §5. 생시 "모름" 처리(시주 제외) 동선 필수.

### 4. 무료 기본 분석 ④ (맛보기 — 신뢰 형성 핵심)
- **Purpose**: 자동 산출로 "진짜 분석했다" 증명 → 결제 유도.
- **공개**: 사주팔자 차트(4열[시·일·월·년]×2행, 일간 셀만 금빛 강조+글로우), 오행 분포 막대(5색, 글로우), 일주 성향 한 줄 카드, 고민 한 줄 답(보라 카드).
- **잠금(티저)**: "올해 운세", "재물·연애·직업" 카드 — `filter:blur(2px)` 스켈레톤 + 금색 자물쇠 SVG + "아직 절반의 이야기가 남아 있어요". 전체의 ~20–30%만 공개.
- **CTA**: 금 그라데이션 "전체 풀이 보기 · ₩6,900".
- 헤드라인 붓글씨 "하늘이 내린 당신의 / 여덟 글자".

### 5. 결제 · ₩6,900 ⑤
- **Purpose**: 마찰 없는 단일가 결제(상품 선택 단계 없음).
- **Layout**: 붓글씨 "마지막 한 걸음" → 주문 요약 카드(사주 리포트 ₩6,900, 금색 가격) → 간편결제 2×2(카카오페이/네이버페이/토스페이/카드, 각 브랜드 색) → [필수] 약관·청약철회 고지 체크 2개 → 환불 고지 박스(앰버 틴트 `rgba(225,193,123,.08)` + `#5A4A2E` 테두리, "제작·발송 전 전액 환불 / 발송 후 제한") → 금 CTA "₩6,900 결제하기".

### 6. 결제 직후 업셀 · ₩19,900 ⑥
- **Purpose**: 객단가 견인(퍼널 수익 핵심). 1회 제안.
- **Layout**: 바텀시트 느낌(외곽 `#07060F` 48px 상단 패딩) → 핸들바 → "단 한 번만 보이는 제안" 보라 배지 → 붓글씨 "방금 본 이야기, / 끝까지 들어보실래요?" → 심화 풀버전 카드(보라 테두리, ◆ 재물·연애·직업운 / 대운 / 맞춤 조언) → 가격(₩29,900 취소선 + **₩19,900** 금색 + "지금만" 배지) → 보라 CTA "네, 풀버전으로 받을게요" → 텍스트 거절 "괜찮아요, 기본만 받을게요" → "거절해도 ₩6,900 기본 리포트는 정상 전달돼요".

### 7. 완료 & 전달 ⑦
- **Layout**: 금빛 체크 원(글로우) → 붓글씨 "결제가 완료됐어요" → 전달 안내 박스(블루 틴트, "3시간 내 카톡으로 발송") → 진행 상태 3단계(결제 완료=금 / 제작 중=보라 글로우 / 발송 완료=빈 원) → 입력 정보 요약 카드 → 재유입 티저(보라 대시 테두리, "기다리는 동안… 이 사주로 궁합도 볼 수 있어요 →").
- 전달 SLA 문구("3시간 내")는 확정값으로 교체.

### 8. 추가 SKU 재유입 ⑧ (LTV 루프)
- **Purpose**: 같은 사주 데이터로 궁합·월별·대운·재회 재판매.
- **Layout**: 붓글씨 "이 사주로 / 더 볼까요?" → **카테고리 탭(전체[금 활성]·연애·재회·재물·궁합)** → SKU 카드 2×2(궁합 ₩9,900[보라 강조·"1차 출시"] / 월별 운세 ₩7,900 / 대운 로드맵 ₩29,900 / 재회·관계 ₩12,900) → 추천 질문 칩("올해 주의할 점은?" 등) → 보라 CTA "궁합 분석 받기 · ₩9,900".

## Interactions & Behavior
- **네비게이션**: 랜딩 CTA → 카카오 로그인 → 입력 → 무료 분석 → (CTA)결제 → 업셀(수락/거절 모두 완료로) → 완료 → 재유입 카드/추천질문 → 해당 SKU 입력·결제 루프.
- **카카오 로그인**: OAuth. 닉네임·이메일 자동 수신. 재방문 시 이전 입력 프리필(입력 화면 스킵 가능).
- **무료 분석**: 차트·오행은 만세력 계산으로 즉시 자동 렌더. 잠금 카드는 blur 처리로 호기심 유지. 결제 후 잠금 해제.
- **업셀**: 결제 성공 직후 1회 모달/시트. 수락 → 심화 추가 결제. 거절 → 기본 상품 정상 진행. 재노출 금지.
- **모션**: 120–240ms ease-out. 페이드/짧은 슬라이드. 바운스 없음. 별 반짝임 `twinkle` 4s ease-in-out infinite(0.25↔0.9 opacity).
- **Hover/press**(데스크톱·웹): 표면 5% 다크 오버레이, press scale(0.98). 금/보라 솔리드 버튼은 화이트 오버레이로 밝게.
- **폼 검증**: 성별·생년월일·동의 필수. 음력 선택 시 윤달 체크 노출. 시각 "모름" 선택 시 시주 제외 안내. 고민은 선택이되 강하게 유도.
- **상태**: 결제 로딩/성공/실패, 제작 진행 상태(결제완료→제작중→발송완료), 환불 가능 여부(발송 전/후) 분기.
