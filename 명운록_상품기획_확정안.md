# 명운록 상품 기획 — 최종 확정안 (잘 팔리는 구조)

> 결정: **고민별 재편 + 무료 갈증 끊기.** 등급제(간명/심명) 폐기, 고민 언어 상품으로 전환.
> 4050이 "내 고민"으로 사게 만들고, 가격 점프를 부드럽게, 무료는 갈증만 남긴다.

---

## 1. 최종 상품 라인업 (6개)

| # | 상품명 (손님이 보는 이름) | 가격 | slug | 역할 |
|---|---|---|---|---|
| 0 | 무료 명식 | ₩0 | (free) | 미끼 — 갈증만 |
| 1 | **내 사주 기본 풀이** | **₩9,900** | basic-saju | 메인 진입. 모든 입구의 종합 1장 |
| 2 | **내 재물·돈 흐름 깊이보기** | ₩19,900 | wealth-saju ✨신규 | 고민별 심화 |
| 3 | **내 관계·가족 깊이보기** | ₩19,900 | relation-saju ✨신규(연애상품 개편) | 고민별 심화 |
| 4 | **2026 월별 운 캘린더** | ₩24,900 | monthly-luck ✨신규 | 킬러 상품·재구매 |
| 5 | **인생 종합 풀이 (대운 60년)** | ₩49,900 | premium-saju | 끝판왕 업셀 |

> 가격 흐름: 9,900 → 19,900 → 24,900 → 49,900 **(부드러운 사다리, 5배 점프 제거)**
> ⚠️ "오늘의 운세 ₩4,900"은 **입구에서 제거** → 결제 고객 전용 보너스/재방문용으로만(앵커 보호). 또는 보류.

---

## 2. 상품명 원칙 — 고민 언어로

- ❌ 간명 / 심명 (4050이 못 알아봄)
- ✅ "내 재물·돈 흐름 깊이보기", "내 관계·가족 깊이보기" (들어온 고민 그대로)
- 한자(命/簡/深/推)는 **장식 배지로만**. 버튼·상품명은 쉬운 한국어.
- 결과지 하단 업셀은 손님이 고른 고민과 **이름이 일치**해야 클릭됨.
  (재물 고민 선택자 → "내 재물·돈 흐름 깊이보기"가 1순위로 노출)

---

## 3. 무료 결과 — "갈증나게 끊기" (대안 C)

### 보여줄 것 (갈증 유발)
1. 내 사주 8글자 ✅ 전체
2. 일간 중심 기운 ✅ 전체
3. 오행 균형 막대 ✅ 전체
4. 강한 기운 / 부족한 기운 ✅ 전체
5. 올해 흐름 **한 줄만** ✅

### 끊을 것 (결제 유도)
6. **선택한 고민 맛보기** → **첫 1~2문장만** 보여주고 끊기
   - 예: "당신의 재물은 한 번에 크게 버는 구조가 아니라…" → **"…(이어서)"** 로 흐림 처리
   - 버튼: "내 재물 흐름 끝까지 보기 · 기본 풀이 ₩9,900"
7. 조심할 시기 / 행동 조언 → **무료엔 없음** (유료 전용)

> 핵심: 무료는 "내 얘기 같다"까지만. "그래서 어떻게?"의 답은 **결제 후**.

## 4. 상품별 결과지 목차

### ① 내 사주 기본 풀이 — ₩9,900 (메인)
약 A4 1장(1,000~1,300자)
1. 내 사주의 핵심 구조 (8글자 큰 그림)
2. 내 중심 기운 (일간) — 어떤 사람인가
3. 성향과 반복되는 패턴 (돈·일·관계)
4. 올해(2026) 전체 흐름
5. **선택한 고민 해설 ★** (가장 깊게)
6. 조심해야 할 시기
7. 지금 하면 좋은 행동 3가지
→ 끝에서: "이 고민, 더 깊게 → 재물/관계 깊이보기" + "인생 전체 → 종합 풀이" 다리

### ② 내 재물·돈 흐름 깊이보기 — ₩19,900
약 A4 1.5장
1. 내 사주 속 재물 구조 (재성·식상 근거)
2. 돈이 들어오는 길 / 새는 길
3. 모으는 힘 vs 쓰는 힘의 균형
4. 올해 재물 흐름 + 큰돈 들어올/조심할 시기
5. 주거·가족·건강 지출과의 연결 (4050 현실)
6. 재물을 지키는 구체적 행동

### ③ 내 관계·가족 깊이보기 — ₩19,900 (기존 연애 개편)
약 A4 1.5장
1. 내 관계 그릇 — 사주로 본 관계 성향
2. 반복되는 관계 패턴 (부부·연애·재혼)
3. 자녀·가족과의 거리·갈등 구조
4. 나와 맞는 사람 / 조심할 사람
5. 올해 관계운 + 시기
6. 관계를 풀어가는 조언

### ④ 2026 월별 운 캘린더 — ₩24,900 (킬러)
약 A4 2장 + 표
1. 올해 전체 큰 흐름 한눈에
2. **1~12월 월별 운** (좋은 달/조심할 달 표시)
3. 큰 결정(이직·계약·이사) 하기 좋은 시기
4. 조심해야 할 달과 이유
5. 분기별 행동 가이드
→ 두고두고 보는 상품 → 만족도·내년 재구매

### ⑤ 인생 종합 풀이 (대운 60년) — ₩49,900 (끝판왕)
약 A4 3~4장
1. 사주 전체 구조 종합
2. 타고난 기질과 그림자
3. 재물운 / 4. 직업·사업운 / 5. 관계·가족운 / 6. 건강 흐름
7. **대운 흐름 (인생 큰 시기별)**
8. 2026 세운 + 향후 3년
9. 시기별 기회와 위기
10. 인생 종합 조언
→ 만세력 API의 대운·세운 데이터 적극 활용

## 5. 업셀·크로스셀 (결과지 하단, 최대 3카드)

- **기본 풀이(₩9,900) 본 사람**, 고른 고민에 따라:
  - 재물·노후·직장 고민 → [推] **내 재물·돈 흐름 깊이보기** ₩19,900
  - 부부·자녀·가족 고민 → [推] **내 관계·가족 깊이보기** ₩19,900
  - 올해 운 고민 → [推] **2026 월별 운 캘린더** ₩24,900
  - 공통 마지막 카드 → **인생 종합 풀이** ₩49,900
- 재입력 없는 원클릭 재구매. 개인화 배지("당신이 궁금해한 재물").

## 7. 추천 진행 순서

1. **기본 풀이(₩9,900)** 목차·톤 확정 → prompt.ts → 데모 검증 ← 지금 시작
2. **무료 갈증 끊기** (맛보기 첫 문장만)
3. 신규 상품 3개 seed 추가 → 각 prompt.ts 목차
4. 크로스셀 매칭 갱신
5. 결과지 **화면 디자인** 입히기

## 8. 솔직한 리스크 / 체크포인트

- **신규 상품 = 만세력 API 호출 1회씩 차감** (6,000회 한도). 상품 늘면 소진 빨라짐 → 키 한도 먼저 확인.
- **월별 운 캘린더**는 12개월치라 LLM 토큰↑ → 생성 비용·시간 늘어남. 가격(₩24,900)으로 충분히 커버됨.
- **상품 6개는 4050에게 많을 수도** → 랜딩엔 "기본 풀이" 하나만 강조하고, 나머지는 결과지 후 업셀로만 노출(선택지 과부하 방지).
- 가격은 시장 보며 조정 가능 (19,900/24,900은 제안값).
